JavaScript关键字（仅有内置API）



变量与声明
let          - 块级作用域可变变量
const        - 块级作用域只读常量
var          - 函数作用域变量（旧版语法）



数据类型与字面量
String       - 字符串类型
Number       - 数字类型
Boolean      - 布尔类型
Null         - 空值对象
Undefined    - 未定义状态
Symbol       - 唯一标识符类型
BigInt       - 大整数类型
"string"     - 表示文本数据（双引号字面量）
'string'     - 表示文本数据（单引号字面量）
`text`       - 模板字符串（反引号字面量）
${var}       - 模板字符串中引入变量的占位符
123          - 整数数字字面量
3.14         - 浮点数数字字面量
true         - 布尔值真
false        - 布尔值假



复杂数据结构
[item1]      - 创建一个按顺序排列的数组
{key: value} - 创建一个包含键值对的对象



核心运算符
+            - 加法或字符串拼接
-            - 减法运算
*            - 乘法运算
/            - 除法运算
%            - 取模求余数
**           - 幂运算
=            - 变量赋值
+=           - 加法赋值
-=           - 减法赋值
==           - 隐式转换相等比较
===          - 严格相等比较
!=           - 不相等比较
!==          - 严格不相等比较
>            - 大于比较
<            - 小于比较
>=           - 大于等于比较
<=           - 小于等于比较
&&           - 逻辑与
||           - 逻辑或
!            - 逻辑非
? :          - 条件三元运算符
??           - 空值合并运算符
?.           - 可选链运算符



控制流与循环
if           - 条件判断
else         - 条件分支不满足时的退路
else if      - 多重条件判断
switch       - 条件分支选择
case         - 分支匹配项
default      - 默认分支
for          - 计数循环
while        - 条件满足循环
do while     - 先执行后判断循环
for in       - 遍历对象属性名（键名）
for of       - 遍历可迭代对象值（元素值）
break        - 立即跳出当前整个循环或 switch 语句
continue     - 跳过本次单次循环，直接进入下一次



函数
function     - 声明普通/具名函数
return       - 返回函数计算结果并终止函数执行
=>           - 箭头函数符号
arguments    - 函数接收的实参伪数组



对象与类
this         - 当前执行上下文对象
new          - 实例化构造函数或类，创建一个新对象
class        - 声明类（构造函数的语法糖）
constructor  - 类的构造函数，在实例化时自动触发
extends      - 实现类与类之间的继承
super        - 调用父类构造函数或父类的方法
static       - 声明类静态属性或方法
get          - 声明属性读取器
set          - 声明属性设置器



现代特性与解构
{a, b} = obj - 对象解构，快速提取对象属性为独立变量
[a, b] = arr - 数组解构，快速提取数组元素为独立变量
...          - 展开/剩余运算符（用于复制、合并或接收剩余参数）



数组方法
push         - 尾部添加元素
pop          - 尾部删除元素
shift        - 头部删除元素
unshift      - 头部添加元素
splice       - 增删改数组指定位置元素
slice        - 截取子数组
concat       - 合并数组
join         - 将数组元素拼接为字符串
indexOf      - 查找元素首次出现索引
includes     - 判断是否包含某个元素
forEach      - 遍历数组每个元素
map          - 映射出新数组
filter       - 过滤符合条件的元素
reduce       - 累加归纳数组元素
find         - 查找第一个符合条件的元素
findIndex    - 查找第一个符合条件元素的索引



字符串方法
length       - 获取字符串长度
charAt       - 获取指定位置字符
substring    - 截取字符串片段
split        - 将字符串分割为数组
replace      - 替换匹配的字符串
replaceAll   - 替换所有匹配的字符串
toLowerCase  - 转换为小写
toUpperCase  - 转换为大写
trim         - 去除两端空格



数学与数字
Math.random  - 生成 0 到 1 之间的随机小数
Math.floor   - 向下取整（去头留尾）
Math.ceil    - 向上取整（有余进一）
Math.round   - 四舍五入取整
Math.max     - 返回一组数中的最大值
Math.min     - 返回一组数中的最小值
parseInt     - 将字符串解析为整数
parseFloat   - 将字符串解析为浮点数
toFixed      - 保留指定位数的小数并转为字符串



异步与错误处理
try          - 尝试执行可能出错的代码块
catch        - 捕获并处理错误的代码块
finally      - 无论是否出错都执行的代码块
throw        - 手动抛出一个自定义的错误对象
Error        - 创建错误对象
Promise      - 异步操作管理对象
then         - 异步成功回调
async        - 声明异步函数（返回一个 Promise）
await        - 暂停异步函数执行，等待 Promise 结果



模块化
export       - 导出当前文件中的变量、函数或类
import       - 导入其他文件导出的变量、函数或类
default      - 默认导出标识符

HTML5关键字



格式化 (Formatting)
 <acronym>    - HTML5 不支持。使用 <abbr> 代替。定义首字母缩写
 <abbr>       - 定义缩写或首字母缩略词
 <address>    - 地址组件
 <b>          - 粗体字
 <bdi>        - 定义范围内文字的方向
 <bdo>        - 改变文字显示方向
 <big>        - HTML5 不支持。使用 CSS 语法代替。大字体
 <blockquote> - 长段区块文字
 <center>     - HTML5 不支持。使用 CSS 语法代替。将对象居中
 <cite>       - 使用引文
 <code>       - 电脑程序内文
 <del>        - 删除的文字
 <dfn>        - 表示术语的定义实例
 <em>         - 斜体字
 <font>       - HTML5 不支持。使用 CSS 语法代替。设置字体大小、字型、颜色
 <i>          - 斜体字
 <ins>        - 定义已插入到文档中的文字
 <kbd>        - 定义键盘输入文字
 <mark>       - 定义标记区的文字
 <meter>      - 产生设定范围的比例图标
 <pre>        - 文字预设样式
 <progress>   - 表现目前工作进度
 <q>          - 内文短引用语
 <rp>         - 定义在不支持 ruby 注释的浏览器中显示的内容
 <rt>         - 定义与显示东亚文字里的声调或注音解释
 <ruby>       - 定义 ruby 注释 (对于东亚文字)
 <s>          - 文字删除线
 <samp>       - 电脑程序样本
 <small>      - 文字缩小
 <strike>     - HTML5 不支持。使用 <del> 或 <s> 代替。文字删除线
 <strong>     - 文字加粗
 <sub>        - 下标字
 <sup>        - 上标字
 <time>       - 定义日期与时间
 <tt>         - HTML5 不支持。使用 CSS 语法代替。电报字格式
 <u>          - 文字下加下划线
 <var>        - 变量或用户定义的文字
 <wbr>        - 定义连续字母中可换行处



 表单和输入 (Forms and Input)
 <form>       - 产生输入表单
 <input>      - 输入栏
 <textarea>   - 多行文字的区域
 <button>     - 按键
 <select>     - 下拉列表
 <optgroup>   - 在下拉列表中定义一组相关选项
 <option>     - 定义下拉列表的参数
 <label>      - 表单控制标签
 <fieldset>   - 设置表单元件组
 <legend>     - 定义 <fieldset> 元素的标题
 <datalist>   - 输入栏的预定义选项列表
 <keygen>     - 标签定义生成密钥
 <output>     - 定义计算结果



 框架 (Frames)
 <frame>      - HTML5 不支持。框架
 <frameset>   - HTML5 不支持。框架组
 <noframes>   - HTML5 不支持。不使用框架
 <iframe>     - 插入行内框架



 图片 (Images)
 <img>        - 插入图片
 <map>        - 影像地图
 <area>       - 定义影像地图中的区域
 <canvas>     - 配合 JavaScript 绘制图案
 <figcaption> - 定义 <figure> 元素的标题
 <figure>     - 定义图片、影像媒体等
 <picture>    - 定义多个图像资源的容器



 音频视频 (Audio and Video)
 <audio>      - 使用音频
 <source>     - 定义媒体来源
 <track>      - 定义 <video> 与 <audio> 之文字内容
 <video>      - 使用视频



 链接 (Links)
 <a>          - 超链接 / 锚点
 <link>       - 链接外部资源
 <nav>        - 定义导航栏链接



 清单 (Lists)
 <ul>         - 无序清单
 <ol>         - 有序清单
 <li>         - 条列项目
 <dir>        - HTML5 不支持。使用 <ul> 代替。定义目录列表
 <dl>         - 定义描述列表
 <dt>         - 在描述列表中定义术语/名称
 <dd>         - 在描述列表中定义术语/名称的描述
 <menu>       - 定义命令列表
 <menuitem>   - 定义用户可以通过弹出菜单调用的命令项



 表格 (Tables)
 <table>      - 插入表格
 <caption>    - 表格标题
 <th>         - 表格之标题单元格
 <tr>         - 表格的列
 <td>         - 表格之单元格
 <thead>      - 表格之标题
 <tbody>      - 表格内文
 <tfoot>      - 表格注解
 <col>        - 表格内之「栏」属性
 <colgroup>   - 群组之表格的「栏」



 风格和语义 (Styles and Semantics)
 <style>      - 使用样式
 <div>        - 文件内区段
 <span>       - 设置行内样式
 <header>     - 定义标题区
 <footer>     - 定义结尾区
 <main>       - 指定文件的主要内容
 <section>    - 定义文件里的区块
 <article>    - 定义文章
 <aside>      - 定义侧栏
 <details>    - 定义互动面板的内容细节
 <dialog>     - 定义对话框或窗口
 <summary>    - 定义互动面板的标签
 <data>       - 链接给定的内容与其机器可读的翻译



 Meta 信息 (Meta Info)
 <meta>       - 网页文件信息
 <base>       - 文件内 URL 之预设位置
 <basefont>   - HTML5 不支持。使用 CSS 语法代替。基本字型样式



 程序设计 (Programming)
 <script>     - 使用语法 (例如 JavaScript)
 <noscript>   - 为不支持客户端脚本语法的用户定义备用内容
 <applet>     - HTML5 不支持。使用 <embed> 或 <object> 代替。插入 JAVA 元件
 <embed>      - 定义外部（非 HTML）应用程序的容器 (嵌入元素)
 <object>     - 嵌入（多媒体）对象
 <param>      - 定义对象的参数
